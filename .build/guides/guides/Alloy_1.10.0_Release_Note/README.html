Alloy 1.10.0 - 25 October 2017

Alloy 1.10.0 is a patch release that includes one improvement.

As of this release, Alloy 1.9.x will not be supported six months from 1.10.0's release date. See Axway Appcelerator Deprecation Policy and Nominal Lifetimes documents for details.

Improvement

  • ALOY-1312 : Upgrade Alloy to support ES6

    • Migrated from using uglifyjs to parse and transform user and library code to using babel and babili
