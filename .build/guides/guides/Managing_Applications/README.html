Managing Applications

To manage an application, click the Apps menu and select it from the list of favorite applications, search for it by name or GUID, or select All to view a list of all applications for the currently selected organization.

You can manage the following types of applications:

  • Client Applications – Titanium, Android or iOS application

  • Mobile Backend Services Datasources (or legacy ACS application) – Only standalone Mobile Backend Services (MBS) datasources (not created with a client application). For MBS datasources created with a client application, access the MBS datasource using the client application's API Builder tab.

  • API Builder Applications
