Alloy 1.13.1 - 10 August 2018

Alloy 1.13.1 is a patch release that includes one bug fix.

As of this release, the previous Alloy patch release is no longer supported. End of support for this version will be 2019-02-10 or until the next patch release. Note: major and minor releases continue to be supported according to their nominal lifetime. See Axway Appcelerator Deprecation Policy and Nominal Lifetimes documents for details.
