Axway Appcelerator Studio Release Notes

Studio 5.1.1.GA - 10 September 2018

Studio 5.1.1.GA is a patch release that includes one bug fix.

As of this GA release, the previous Studio patch release is no longer supported. End of support for this version will be up to 2019-03-06 or until the next patch release. Note: major and minor releases continue to be supported according to their nominal lifetime. See Axway Appcelerator Deprecation Policy and Nominal Lifetimes documents for details.

Fixed issue

  • TISTUD-9113 - Studio's method for picking up release notes needs updating
